# football-video-learning

Repositorio de aprendizaje para procesar un vídeo de un partido de fútbol.
El objetivo es aprender paso a paso, desde cero, cómo:
- Guardar la URL del vídeo origen
- Descargar el vídeo manualmente más adelante
- Cargarlo en Python
- Leer frames y entender su estructura

De momento, solo estamos preparando el proyecto.
